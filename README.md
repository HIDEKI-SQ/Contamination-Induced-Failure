# Contamination-Induced Failure (CIF) Phase Structure

**Author:** HIDEKI (Independent Researcher, Japan)  
**ORCID:** 0009-0002-0019-6608  
**Paper:** "Backfire Phase Structure in Contaminated Chain-of-Thought Reasoning" (submitted, 2026)

## Overview

When language models consume chain-of-thought reasoning generated by other models, errors in the consumed trace can cause the consumer to abandon its own correct reasoning. We term this phenomenon **contamination-induced failure (CIF)**.

This repository contains the experimental data, analysis code, and reproducibility materials for a systematic study of CIF across five language models, three reasoning domains, and four contamination levels.

## Key Results

- **CIF rates:** 1% to 86% across 15 model-domain cells (n=100)
- **Gap amplification:** Up to 57x between models on the same task
- **Sycophancy dissociation:** The most socially compliant model shows the lowest CIF

## Reproducibility

All experiments were conducted via API calls to:
- Anthropic API: Claude Sonnet 4, Claude Haiku 3.5
- OpenAI API: GPT-4o, GPT-4o-mini, GPT-3.5-turbo

Experiments are fully deterministic (temperature=0, fixed seeds). Running the notebooks requires API keys set as Colab Secrets.
